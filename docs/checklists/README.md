# 체크리스트

이 디렉터리는 적용 전에 확인해야 할 기준을 모은다.

## 문서 목록

| 문서 | 용도 |
|------|------|
| `pre-apply.md` | apply 전 공통 점검 |
| `module-review.md` | 모듈 리뷰 체크리스트 |

## 어떻게 쓰나

1. `plan`을 만든 뒤 `pre-apply.md`를 먼저 본다.
2. 새 모듈이나 큰 변경은 `module-review.md`를 통과시킨다.
3. prod 반영 전에 두 문서를 다시 확인한다.
